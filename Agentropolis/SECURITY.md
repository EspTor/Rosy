# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 0.1.x   | ✅ |
| < 0.1   | ❌ |

## Reporting a Vulnerability

**Do not** report through public GitHub issues.

Email security@agentropolis.org with:
- Description
- Steps to reproduce
- Potential impact
- Mitigation suggestions

We will respond within 48h acknowledging, 7d for detailed response.

## Security Best Practices

- Keep Docker images updated
- Use strong unique passwords
- Restrict API access via firewall
- Enable TLS everywhere
- Rotate JWT secrets every 90 days
- Monitor audit logs
- Use rate limiting
