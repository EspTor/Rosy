# Glossary

## Core Terms

**Agent**: Autonomous entity with identity, goals, actions.
**Agentropolis**: The virtual town simulation.
**Tick**: Discrete simulation time step (e.g., 100ms real).
**Town**: 2D spatial environment (1000×1000 grid).

## Agent Anatomy

**BODIES**: Brain, Body, Social graph, Identity framework.
**Perception**: What agent senses (nearby agents, objects, resources).
**Goal**: Something agent wants (hierarchical: life → tactical).
**Memory**: Episodic (events), semantic (facts), procedural (skills).
**Personality**: OCEAN traits + extras (impulsivity, risk tolerance).

## Economy

**Transaction**: Resource transfer (buy/sell/gift).
**Supply/Demand**: Price formation mechanism.
**Gini Coefficient**: Wealth inequality measure (0=equal, 1=unequal).
**Business**: Entity engaging in economic activity.
**Property**: Ownable assets (real estate, goods).

## Interactions

**Relationship**: Social bond with trust/affection/familiarity scores.
**Conversation**: Multi-turn dialog with emotional tone.
**Collaboration**: Multiple agents working together.
**Conflict**: Adversarial interaction (competition, crime).

## Technical

**ECS**: Entity-Component-System architecture.
**A***: Pathfinding algorithm.
**JWT**: JSON Web Token for authentication.
**WAL**: Write-Ahead Logging (database durability).
**Vector DB**: Database optimized for similarity search (memories).
