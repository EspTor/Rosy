# Project Charter

## Mission

Create an open, extensible simulation platform where AI agents can live, interact, and evolve.

## Vision

Agentropolis becomes the standard benchmark for studying coordinated AI behavior.

## Goals

1. Research platform for reproducible multi-agent experiments
2. Emergence observatory with analysis/visualization tools
3. Extensible architecture for community contributions
4. Educational tool for AI concepts

## Success Metrics (12 months)

- 500+ agents at 20 ticks/sec
- 20 external AI brains connected
- 3 published papers citing Agentropolis
- 1000+ GitHub stars
- 50+ external contributors

## Scope

**In**: Core engine, brain API, persistence, dashboard, basic economy, social dynamics, security.

**Out** (v1.0): Advanced 3D graphics, voice, physical realism, commercial hosting, mobile app.

## Stakeholders

Creator: E T; Contributors: open source community; Users: researchers, educators

## Risks & Mitigations

- LLM costs → cache, rate-limit, smaller models
- Performance bottlenecks → profiling, parallelization
- Security vulnerabilities → audit, fuzzing, bug bounty
- Lack of uptake → outreach, demos
- Developer burnout → share maintainership

## Milestones

M1 Core prototype (10 agents, 10 ticks/sec) - Month 2
M2 Alpha release - Month 4
M3 Social dynamics - Month 6
M4 External brains - Month 8
M5 Beta release - Month 10
M6 1.0 release - Month 12
